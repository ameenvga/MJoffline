മലയാളം യൂനികോഡില്‍ ടൈപ്പ് ചെയ്യാനുള്ള ഒരു ടൂള്‍ ആണ് മലയാള ജാലകം‌.‌ വ്യത്യസ്ത കീബോര്‍ഡുകള്‍ ഉപയോഗിച്ച് ടൈപ്പ് ചെയ്താലും നിങ്ങള്‍ക്ക് ലഭിക്കുന്നത് യൂനികോഡ് ഫോര്‍മാറ്റിലുള്ള മലയാളമാണ്. ഇന്റര്‍നെറ്റ് ഉപയോക്താക്കള്‍ക്ക് ഈ ടെക്സ്റ്റ് നേരിട്ടു തന്നെ ഫെയ്സ്ബുക്ക് പോലെയുള്ള സമൂഹ മാധ്യമങ്ങളില്‍ പേസ്റ്റ്‌ ചെയ്ത്‌ ഉപയോഗിക്കാവുന്നതാണ്.

മലയാളത്തിനു‌ പുറമേ‌ ഇംഗ്ലീഷിലും‌ അറബിയിലും‌ നിങ്ങള്‍ക്ക്‌ ടെക്സ്റ്റ്‌ എന്റര്‍‌ ചെയ്യാവുന്നതാണ്‌. ഇംഗ്ലീഷ്‌, മലയാളം‌‌, അറബി‌ ഭാഷകള്‍‌ ‌യഥാക്രമം‌ F1, F2, F3 ( മാക് കീബോര്‍ഡില്‍‌ Command k‌ey + 1, Command ‌key + 2, Command key + 3 എന്നിങ്ങനെ‌ ) എന്നീ‌ ഷോര്‍ട്ട്‌കട്ടുകള്‍‌ ഉപയോഗിച്ച്‌ സെലക്ട്‌ ചെയ്യാവുന്നതാണ്‌. അല്ലെങ്കില്‍‌ മുകളിലെ‌ ബാറിലെ‌ മൂന്ന്‌ ബട്ടണുകളില്‍‌ ക്ലിക്ക്‌ ചെയ്തും‌‌ ഇഷ്ടമുള്ള‌ ഭാഷ‌ സെലക്ട്‌ ചെയ്യാവുന്നതാണ്‌.

മലയാള ഭാഷയില്‍ ടൈപ്പ് ചെയ്യാന്‍ മൂന്ന് കീബോര്‍ഡുകള്‍ ലഭ്യമാണ്. ടോപ്ബാറില്‍ കാണുന്ന ആദ്യത്തെ ഒപ്ഷനില്‍ ക്ലിക്ക് ചെയ്താല്‍ വ്യത്യസ്ത കീബോര്‍ഡുകള്‍ സെലക്ട് ചെയ്യാന്‍ കഴിയും. അവയുടെ കീമാപ്പ് കാണാന്‍ സൈഡ് ബാറിലുള്ള കീബോര്‍ഡ് ഐക്കണില്‍ ക്ലിക്ക് ചെയ്താല്‍ മതി. കീമാപ്പ് താഴെ പ്രത്യക്ഷപ്പെടും.

അറബി ഭാഷയില്‍ ടൈപ്പ് ചെയ്യാന്‍ രണ്ട് തരം കീബോര്‍ഡുകള്‍ ലഭ്യമാണ്. അവ ലഭിക്കാന്‍ ആദ്യം അറബി ഭാഷ സെലക്ട് ചെയ്ത് കീബോര്‍ഡ് മെനുവില്‍ ക്ലിക്ക് ചെയ്താല്‍ മതി. അവയുടെയും കീമാപ്പ് സൈഡ്ബാറിലെ കീബോര്‍ഡ് ബട്ടണില്‍ ക്ലിക്ക് ചെയ്ത് കാണാം.

അറബി ഭാഷ, വലതു ഭാഗത്ത് നിന്നും ഇടത് ഭാഗത്തേക്ക് എഴുതുന്ന ഭാഷയായതിനാല്‍ ടൈപ്പ് ചെയ്യുമ്പോള്‍, ടെക്സ്റ്റ് ഏരിയയിലെ ടെക്‌സ്റ്റ് അലൈന്‍‌മെന്റ് ക്രമം തെറ്റി കാണപ്പെടും. എങ്കിലും തിരികെ മലയാളത്തിലേക്കോ ഇംഗ്ലീഷിലേക്കോ ടൈപ്പിങ്ങ് മാറുമ്പോള്‍ അലൈന്‍‌മെന്റ് ശരിയായി വരുന്നത് കാണാം.

അറബിയില്‍ മാത്രം ടൈപ്പ് ചെയ്യുന്നവര്‍ക്ക് മുകളിലെ Alignment menu വില്‍ നിന്ന് Right Alignment സെലക്ട് ചെയ്ത് ടൈ‌പ്പ് ചെയ്യുന്നതാണ്‌ ഏറ്റവും‌ നല്ലത്‌.

ടെക്സ്റ്റ്‌ അലൈന്മെന്റ്‌ ഓപ്ഷനുകള്‍‌ ഉപയോഗപ്പെടുത്തി‌ ടെക്സ്റ്റിനെ‌ ഇടത്‌ ഭാഗത്തോ‌ വലതു‌ ഭാഗത്തോ‌ മധ്യഭാഗത്തോ‌ ഒക്കെയായി‌ അറേഞ്ച്‌ ചെയ്യാവുന്നതാണ്‌.

ടോപ് ബാറിലെ പ്ലസ് മൈനസ് ബട്ടണുകളില്‍ ക്ലിക്ക് ചെയ്ത് ഫോണ്ട് സൈസുകള്‍ കൂട്ടുകയോ കുറക്കുകയോ ചെയ്യാം. Ctrl + , Ctrl - ഷോര്‍ട്ട്കട്ടുകളും ഉപയോഗിച്ച് ഇതേ ഓപ്പറേഷന്‍ നിര്‍വ്വഹിക്കാം.

ടെക്സ്റ്റ്‌ കോപ്പി‌ ചെയ്യാനുള്ള‌ ഒപ്ഷന്‍‌ ഉപയോഗിച്ച്‌ മുഴുവന്‍‌ ടെക്സ്റ്റും‌ ഒറ്റ‌ ക്ലിക്കില്‍‌ കോപ്പി‌ ചെയ്യാന്‍‌ കഴിയും‌‌. ഇനി‌ സെലക്ട്‌ ചെയ്ത‌ ശേഷമാണെങ്കില്‍‌ സെലക്ട്‌ ചെയ്ത‌ ഭാഗം‌ മാത്രമാണ്‌ ക്ലിപ്പ്‌ ബോര്‍ഡില്‍‌ കോപ്പി‌ ആവുക‌.‌

U‌ndo Red‌o ബട്ടണുകള്‍‌ ഉപയോഗിച്ച്‌ നിങ്ങളുടെ‌ ഓപ്പറേഷനുകളില്‍‌ മുന്നിലേക്കും‌ പിന്നിലേക്കും‌ പോകാവുന്നതാണ്‌.

FML, ML-KV ഫോര്‍മാറ്റുകളിലേക്ക് യൂനികോഡ്‌ ടെക്സ്റ്റ്‌ കണ്‍വെര്‍ട്ട് ചെയ്യാനുള്ള ഒപ്ഷന്‍ Convert to എന്ന‌ മെനുവില്‍‌ ലഭ്യമാണ്‌. നിങ്ങളുടെ‌ ടെക്സ്റ്റില്‍‌ ഇംഗ്ലീഷ്‌ വാക്കുകളുണ്ടെങ്കില്‍‌ അത്‌ പ്രത്യേകമായി‌ തന്നെ‌ കണ്‍വേര്‍ട്ടഡ്‌ ബോക്സില്‍‌ കാണിക്കും‌. ഫോട്ടോഷോപ്പ്‌ പോലെ‌യുള്ള‌ മറ്റു‌ സോഫ്റ്റ്‌വെയറുകളിലേക്ക്‌ പേസ്റ്റ്‌ ചെയ്യുമ്പോള്‍‌ ഇംഗ്ലീഷ്‌ വാക്കുകളെ‌ പ്രത്യേകമായി‌ ശ്രദ്ധിക്കാ‌നാണ്‌ ഈ‌ സൗകര്യം‌.
ടെക്സ്റ്റുകള്‍ FML, MLKV ഫോര്‍മാറ്റുകളിലേക്ക് മാറ്റി ക്ലിപ്പ്ബോര്‍ഡിലാക്കാന്‍ Copy to FML ( Ctrl F , Copy to MLKV Ctrl M എന്നീ ഒപ്ഷനുകള്‍ ഉപയോഗിക്കാം.

അക്ഷരങ്ങള്‍ തൊട്ടുമുമ്പിലെ അക്ഷരങ്ങളുമായി ചേര്‍ന്ന് വരാതെ വേണമെന്നുണ്ടെങ്കില്‍ ഒരു vertical bar | (കീ‌ബോര്‍ഡില്‍‌ എന്റര്‍‌ ബട്ടണു‌ മുകളില്‍‌ ) ഇന്‍സര്‍ട്ട് ചെയ്താല്‍ മതി. അപ്പോള്‍ പ്രത്യക്ഷമായി ഒന്നും സംഭവിക്കില്ല പക്ഷെ, മുന്‍‌ അക്ഷരത്തിനും ഇനി ടൈപ്പ് ചെയ്യാന്‍ പോകുന്ന അക്ഷരത്തിനുമിടയില്‍ അദൃശ്യമായ ഒരു ബ്രെയ്ക് ചേര്‍ക്കപ്പെടും. എന്നാല്‍ സ്പേസിനു ശേഷ‌മാണ്‌ vertical bar‌ ബട്ടണ്‍‌ പ്രസ്‌ ചെയ്യുന്നതെങ്കില്‍‌ അവിടെ‌ ‌vertical bar തന്നെയാണ്‌ ഇന്‍സര്‍ട്ട്‌ ചെയ്യപ്പെടുക‌.

FML, MLKV, ML ഫോര്‍മാറ്റിലുള്ള ടെക്സ്റ്റുകള്‍ Paste from ഒപ്ഷനുകള്‍ ഉപയോഗപ്പെടുത്തി പേസ്റ്റ് ചെയ്യുമ്പോള്‍ ഇംഗ്ലീഷ് വാക്കുകള്‍ അര്‍ഥരഹിതമായ ചില അക്ഷരക്കൂട്ടങ്ങളായി മാറും. അവയെ പഴയ പടി ഇംഗ്ലീഷിലേക്ക് മാറ്റാന്‍ കോണ്ടെക്സ്റ്റ് മെനുവിലെ To English എന്ന ഒപ്ഷന്‍ ഉപയോഗപ്പെടുത്താം. ഇംഗ്ലീഷിലേക്ക് മാറ്റേണ്ട ടെക്സ്റ്റ് സെലക്ട് ചെയ്ത് Right Click ചെയ്യുക. Ctrl + + - ടെക്സ്റ്റ് ഫോണ്ട് സൈസ് വലുതാക്കാന്‍

Ctrl + - - ടെക്സ്റ്റ് ഫോണ്ട് സൈസ് ചെറുതാക്കാന്‍


Special Codes

Alt + S - ﷺ - സ്വല്ലല്ലാഹു അലൈഹി വസല്ലം

Alt + R - ₹ - ഇന്ത്യന്‍ രൂപയുടെ ചിഹ്നം

Alt + Y - € - യൂറോ കറന്‍സി ചിഹ്നം

Alt + G - © - കോപിറൈറ്റ് ചിഹ്നം

Alt + T - ™ - ട്രേഡ് മാര്‍ക്ക് ചിഹ്നം

Alt + D - ® - രെജിസ്റ്റേര്‍ഡ് ചിഹ്നം
